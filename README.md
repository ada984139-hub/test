# PelețiPro — Magazin demo (fără backend)

Site demo pentru vânzare peleți, construit doar cu HTML, CSS și JavaScript (fără backend).

Fișiere generate:
- `index.html` — pagină principală
- `style.css` — stiluri (responsive, grid/flexbox, animații)
- `script.js` — generare produse, coș simplu în `localStorage`, animații UI

Deschide proiectul local:
- Metoda rapidă: fă dublu-click pe `index.html` pentru a o deschide în browser.
- Recomandat (server local pentru fetch-uri/compatibilitate): deschide un terminal în folder și rulează:

PowerShell:
```powershell
python -m http.server 8000
```
Apoi accesează `http://localhost:8000` în browser.

Modificări rapide:
- Produsele se află în `script.js` în variabila `products`.
